Video Settings

These settings control the way that the lake environment is rendered to the computer screen. Options include display resolution, smooth textures, and smooth shading.

Display Resolution: Select the display resolution (width x height) that you would like the screensaver to run in. Available options will depend upon your specific computer setup. (Higher resolutions will look better but will cause the program to run more slowly.) The default value is 640 x 480.

Smooth Textures: This setting controls the rendering of textures in the environment. Check this box to enable smooth textures. Smooth textures look nicer but will cause the program to run more slowly than “blocky” textures. The default value is “on”.

Smooth Shading: This setting controls the smoothness of the lighting in the environment. Check this box to enable smooth shading. Smooth shading looks nicer but causes the program to run more slowly than “flat” shading. The default value is “on”.
