Environment Settings

These settings control the way that the lake environment is set up and displayed. You have complete control over the numbers of creatures that are displayed as well as environment settings and a small clock, which can be displayed in the lower-right corner of the screen.

Number of Swans: Enter the total number of swans desired (includes both adult and young swans). Valid values are in the range 0-36. The default value is 24.

Number of Turtles: Enter the total number of turtles desired. Valid values are the range 0-8. The default value is 3.

Number of Dragonflies: Enter the total number of dragonflies desired. Valid values are the range 0-16. The default value is 8.

Number of Fish: Enter the total number of jumping fish desired. Valid values are the range 0-8. The default value is 3.

Number of Eagles: Enter the total number of eagles desired. Valid values are the range 0-8. The default value is 2.

Fog/Haze: Check this box the enable rendering of fog/haze effects. These effects may cause the program to run slightly slower. The default value is “on”.

Lens Flares: Check this box to enable rendering of lens flares (the glare of the sun on the screen). This effect may cause the program to run slightly slower. The default value is “on”.

Ripples: Check this box to enable rendering of ripple effects on the surface of the water. (This is not the same as the “shimmering” effect of the water.) This effect may cause the program to run slightly slower. The default value is “on”.

Show Clock: Check this box to enable a small digital clock in the lower-right corner of the display. The default value is “off”.
