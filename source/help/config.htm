Configuration Menu

An easy-to-use configuration menu allows all screensaver options to be set according to your personal preferences. This menu can be accessed by clicking on the “Settings” button in the Windows “Display Settings” dialog box, under the “Screensaver” tab.

The following groups of options are available: Environment Settings, Video Settings, Audio Settings, and Advanced Options. Several buttons are available to perform various actions, as well. These are: Reset Defaults, Purchase (trial version only), Help, OK, and Cancel. All of these settings and functions are described in the sections that follow.

Additionally, clicking on the Swan Lake Screensaver graphic will open the “About” dialog box, which includes software version number and other information. From here, you can also visit the WaveBreak Software website and view the software license agreement and “readme” files.
