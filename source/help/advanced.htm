Advanced Settings

These settings are for advanced users who want more control over how the program operates. Options include running the screensaver application in priority mode and using triple display buffering. Do not change these setting unless you understand what they are for and what they can do.

Run In Priority Mode: Check this box to enable priority-mode handling of the screensaver application. This means that the Windows system will place this program ahead of most other programs that are running in the background. Turning this setting on will allow the screensaver to run more smoothly, but all other background applications will run slower. This setting can cause networking operations to slow down considerably and may cause some other programs to temporarily stop responding. The default value for this setting is “off”.

Use Triple Buffering: Check this box to enable triple display buffering (as opposed to the usual double-buffering). This setting can allow the screensaver to run more smoothly depending on your display adapter and configuration. On some systems, turning this setting on can actually cause the screensaver to run much less smoothly, or can cause it to not run at all. The default value for this setting is “off”.
